리스크. 사이드바 폭이 좁아서 라벨이 본문 위로 떠 있는 모양 — 본문 첫 행과 충돌할 수 있음. 작게/희미하게.

리스크. 도메인마다 1순위 액션이 다르면(메모는 입력, 캘린더는 이동 등) 일관성이 깨질 수 있음.
